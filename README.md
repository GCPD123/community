## 开发遇到的问题
在页面引入bootstrap的js之前一定要引入jq否则有些功能不会生效，引入之后一定要在maven中重新导入 否则引入也不会生效  

## 快捷键
alt + 拖移 = 选中一列快速编辑  
fn + command + f9 = 编译当前文件（配合热部署）  
fn + command + shift + f12 = 全屏当前画面  
command + alt + v = 在方法参数外面快速new对象
